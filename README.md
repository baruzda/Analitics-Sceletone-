# HR-аналитика — CSS Skeleton

Демо скелетона загрузки страницы HR-аналитики (Figma, 394px, адаптив).

## Демо

Откройте [`index.html`](index.html) локально или на GitHub Pages и нажмите **ПОКАЗАТЬ ДЕМО**.

Автозагрузка через 3 с: `index.html?autoload`

## Файлы

| Файл | Описание |
|------|----------|
| `page-skeleton.css` | Скелетон (интегрировать в проект) |
| `demo.css` | Стили loaded-состояния |
| `INTEGRATION.md` | Инструкция для разработчика |

## Локальный запуск

```bash
python3 -m http.server 8877
```

Откройте http://localhost:8877
